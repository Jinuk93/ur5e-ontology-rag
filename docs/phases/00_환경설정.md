## 4. 환경 구성

### 4.1 Python 환경

| 항목 | 값 |
|------|-----|
| Python 버전 | 3.11.9 |
| 가상환경 | venv |
| 패키지 수 | 28+ |

### 4.2 외부 서비스

| 서비스 | 용도 | 설정 |
|--------|------|------|
| Neo4j | 지식그래프 저장소 | Bolt 프로토콜, localhost:7687 |
| OpenAI API | Embedding + LLM | text-embedding-3-small, gpt-4o-mini |
| ChromaDB | 벡터 DB | 로컬 영속 저장 |

### 4.3 폴더 구조

```
ur5e-ontology-rag/
├── src/
│   ├── api/          # FastAPI 서버
│   ├── dashboard/    # Streamlit UI
│   ├── embedding/    # 벡터 임베딩
│   ├── ingestion/    # 데이터 수집
│   ├── ontology/     # 온톨로지/그래프
│   ├── rag/          # RAG 파이프라인
│   ├── sensor/       # 센서 처리
│   └── evaluation/   # 평가 시스템
├── data/
│   ├── raw/          # 원본 데이터 (PDF)
│   ├── processed/    # 처리된 데이터
│   ├── sensor/       # 센서 데이터
│   └── benchmark/    # 평가 데이터
├── stores/
│   ├── chroma/       # ChromaDB 인덱스
│   └── audit/        # 감사 로그
├── configs/          # 설정 파일
├── scripts/          # 실행 스크립트
├── tests/            # 테스트 코드
└── docs/             # 문서
```
