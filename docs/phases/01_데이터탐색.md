## 3. 대상 문서

### 3.1 문서 목록

| # | 문서 | 페이지 | 주요 내용 |
|---|------|--------|----------|
| 1 | UR5e Service Manual | ~300 | 유지보수, 분해/조립, 교정 |
| 2 | UR5e Error Codes | ~50 | 에러코드, 원인, 해결책 |
| 3 | UR5e User Manual | ~200 | 설치, 운영, 안전지침 |

### 3.2 문서 구조 분석

**Service Manual 구조**:
```
1. Safety (안전 지침)
2. Introduction (제품 소개)
3. Unpacking (개봉/설치)
4. Getting Started (시작하기)
5. Hardware (하드웨어)
   - Control Box
   - Robot Arm
   - Joints
6. Maintenance (유지보수)
7. Troubleshooting (문제 해결)
8. Calibration (교정)
```

**Error Codes 구조**:
```
- Error Code: C1xxxx (제어기)
- Error Code: S1xxxx (안전)
- Error Code: J0-J5xxxx (조인트별)
- 각 에러: 코드, 설명, 원인, 조치
```

## 4. 핵심 발견 사항

### 4.1 에러코드 패턴

| 접두사 | 분류 | 예시 |
|--------|------|------|
| C1 | 제어기 에러 | C10000, C15402 |
| S1 | 안전 에러 | S10001, S1A507 |
| J0~J5 | 조인트 에러 | J00001, J5A505 |

### 4.2 주요 컴포넌트

| 컴포넌트 | 영문명 | 관련 에러 |
|----------|--------|----------|
| 제어 박스 | Control Box | C1xxxx |
| 티치 펜던트 | Teach Pendant | C1xxxx |
| 조인트 0~5 | Joint 0~5 | J0~J5xxxx |
| 툴 커넥터 | Tool Connector | Txxxx |
| 베이스 | Base | - |

### 4.3 에러 구조 (예시)

```yaml
error_code: C154A3
message: "Control box fan malfunction"
causes:
  - "Fan failure"
  - "Cable disconnection"
  - "Dust accumulation"
resolutions:
  - "Check fan operation"
  - "Verify cable connections"
  - "Clean dust from fan"
```

## 5. 샘플 QA 데이터

### 5.1 에러코드 질문
